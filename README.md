# 적용 기술
- Python
- Django Web Framework
- Bcrypt
- JWT
- MySQL
- AWS EC2, RDS
- Gunicorn
- CORS header
- Beautifulsoup, Selenium

# 구현 기능
## 계정
- Bcrypt를 사용하여 패스워드 해싱
- JWT를 사용하여 로그인 시 토큰 발행 
- 회원 가입 및 로그인
- 이메일 유효성 검사
- 마이 페이지

## 상품
- 메인페이지
- 상세페이지
- 상품 검색 기능

## 리뷰
- 상품 별로 리뷰 리스트 반환
- 로그인한 사용자만 작성
- 수정 및 삭제 기능 지원
- 평균 평점 및 리뷰 수 데이터 반환
